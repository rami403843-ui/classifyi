# 📢 ClassiFyi — сайт объявлений с фото и ценами

## Идея
ClassiFyi — это онлайн-доска объявлений, где любой пользователь может
зарегистрироваться, выложить товар с фото и ценой, а покупатели — искать
нужные вещи по категориям и цене. Как мини-версия lalafo/OLX.

## Базовый функционал
- Регистрация и вход пользователей
- CRUD объявлений: создать, посмотреть, редактировать, удалить (только владелец)
- Главная страница-каталог с карточками объявлений
- Детальная страница объявления
- Поиск по названию + фильтры: категория, цена от/до
- Загрузка изображений

## Челлендж-функции
- ✅ Статус объявления (активно / продано) — проданные скрыты с витрины
- ✅ Несколько фото на одно объявление
- ✅ Страница «Мои объявления»
- ✅ Счётчик просмотров на каждом объявлении
- ✅ Адаптивный дизайн под телефон

## Технологии
- Python 3.10+ / Django 4.2
- SQLite
- HTML + CSS (свои стили, без фреймворков)

## Как запустить
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
Открыть в браузере: http://127.0.0.1:8000

Тестовые аккаунты:
- пользователь: `test` / `test12345`
- админка (http://127.0.0.1:8000/admin): `admin` / `admin12345`
